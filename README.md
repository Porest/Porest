# Porest
2017 글로코톤 '공감' 주제       
Yeee          

1. 유저 회원가입
2. 유저 로그인
3. 나무 생성
4. 나무 태그, 나무 여부, 나무 수명, 등 나무에 관한 정보 입력
5. 나무는 글 을 쓸 경우 잎이 하나 씩 달리고 성장 한다.
6. 나무의 수명이 끝날경우 보상을 지급
7. 나무를 공유하여서 숲을 만들 수 있다.
8. 숲은 태그 별로나 원하는 사람과 공유를 해 만들 수 있다.
9. 숲의 경우에는 비밀의 숲과 공개된 숲으로 나누어져 있다.
10. 숲과 나무는 검색이 가능하다.
11. 검색 요소는 태그, 일자, 등 이 있다.
